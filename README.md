## Getting started

- Recommended `node js 16.x` and `npm 6+`. (suggestion v16.15.0)
- Install dependencies: `npm install` / `yarn install`
- Start the project: `npm run start` / `yarn start`

## API
- src/utils/config.js
  - server: server url (backend)
  - port : server port
  - baseURL : server api base url
